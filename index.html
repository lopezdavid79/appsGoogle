Curso de Aplicaciones de Google

¡Te damos la bienvenida al curso de Aplicaciones de Google! Este espacio está diseñado para ti, para que puedas potenciar tus habilidades digitales y dominar herramientas clave como Gmail, Drive, Documentos y más. Juntos, exploraremos cómo estas aplicaciones pueden abrirte nuevas puertas en tus estudios y en tu carrera profesional. ¡Estamos muy contentos de que te sumes a este camino de aprendizaje con el Programa Ágora Argentina!

Atajos de Teclado para el Reproductor

Tecla Acción
Espacio o K Reproducir / Pausar el video.
M Silenciar / Activar el sonido.
← / → Retroceder / Avanzar 5 segundos.
J / L Retroceder / Avanzar 10 segundos.
↑ / ↓ Subir / Bajar el volumen.
F Activar / Desactivar la pantalla completa.
C Mostrar / Ocultar los subtítulos (si están disponibles).
Teclas 0 a 9 Saltar a un punto específico del video (del 0% al 90%).
